# CS-405-Secure-Coding
Course projects focused on secure coding, software security, and vulnerability prevention.

CS-405 Secure Coding

This repository contains my work from CS 405: Secure Coding at Southern New Hampshire University.

Throughout this course, I learned how to build more secure software by applying secure coding standards and security best practices throughout the software development life cycle. The projects in this repository demonstrate concepts such as secure coding, static code analysis, unit testing, encryption, vulnerability prevention, security policies, and risk assessment.

Repository contents include:

* Project One Security Policy
* Project Two Presentation
* Coding assignments
* Portfolio reflection

This course reinforced my understanding of software security and strengthened my ability to identify vulnerabilities, evaluate security risks, and develop software with security in mind from the beginning of the design process.
